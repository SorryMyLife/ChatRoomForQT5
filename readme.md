使用qt5实现的一个简易聊天室

功能：聊天时自动显示客户端发送的图片、支持一对多聊天、支持文件、图片传输、支持多个客户端一起互相聊天、支持群聊

依赖：core gui network

qt版本：Qt 5.14.1/QT CREATOR 4.11.1

编译器: mingw73_32
